# Capture Maps UI

Front end for [Capture Maps API](https://github.com/zxcV32/capture-maps-api) to download detailed
Maps (Google Maps)

# Setup
1. Create `.env` in the project root and store Google Maps Static API|

   `REACT_APP_GOOGLE_MAPS_API_KEY=<YOUR API_KEY>`
2. Start Project

   `npm start`
